# Botanical Minimal Design System

A plant-inspired minimalist design language combining Swiss cleanliness, technical precision, and organic warmth.

## How to View

```bash
# From the inspiration directory
open src/designs/index.html
```

## Development

```bash
# Install dependencies
npm install

# Build CSS (one-time)
npm run build:css

# Watch mode for development
npm run watch:css
```

## Pages

- `index.html` - Home/About page
- `writing.html` - Blog listing page
- `projects.html` - Projects showcase page
